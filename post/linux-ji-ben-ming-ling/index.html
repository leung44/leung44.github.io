查看网络连接信息

nestat -ntlp
参数 作用
-a (all)显示所有选项，默认不显示 LISTEN 相关
-t (tcp)仅显示 tcp 相关选项
-u (udp)仅显示 udp 相关选项
-n 拒绝显示别名，能显示数字的全部转化成数字。
-l 仅列出有在 Listen (监听) 的服務状态
-p 显示建立相关链接的程序名
-r 显示路由信息，路由表
-e 显示扩展信息，例如 uid 等
-s 按各个协议进行统计
-c 每隔一个固定时间，执行该 netstat 命令。

文件列表

命令 作用
ls list
ll 列出详细资料
la 列出隐藏资料 
# 当前路径
pwd # print work directory

# 创建文件夹
mkdir # make directory

# 复制
cp source dest # copy
cp -r source dest # 递归带目录复制
cp -r source/. dest # 复制目录下文件

# 移动
mv # move(rename)
mv source dest

# 删除
rm # remove
rm -rf # 递归,强行删除

# 查看文件
cat

# 搜索
cat info.log | grep 'keyword'

#查看倒数几行
tail
tail -f info.log # 滚动显示

# 终止进程
kill pid
kill -9 pid # 强行终止

# 清屏
clear
reset

# 查找
find / -name ''

# 查看端口占用
netstat -aon|findstr 8080
tasklist|findstr ${pid}
taskkill -t -f -pid ${pid}

# 杀死端口进程
kill -9 $(lsof -t -i:8081)

# 授权 shell 脚本
chmod +x *.sh

# 最近登录
last

# 最近失败登录
lastb

开放端口

# 查看是否已开放
firewall-cmd --query-port=9090/tcp

# 若提示 FirewallD is not running, 开启防火墙
systemctl start firewalld

# 开启端口
firewall-cmd --add-port=9090/tcp --permanent

# 重新加载配置
firewall-cmd --reload

# 关闭防火墙
systemctl stop firewalld

# 移除端口
firewall-cmd --permanent --remove-port=9090/tcp

# 查看端口能否远程连接
telnet ip port
