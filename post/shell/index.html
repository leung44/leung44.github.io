HELLOWORLD

#!/bin/sh
echo "Hello World"

"#!"约定标记, 通知系统需要什么解析器来执行
"echo" 输出命令

变量

创建变量

AUTHOR_NAME="TonyLeung"
  • 不用$符号
  • 不能有空格
  • 不能数字开头

使用变量

$AUTHOR_NAME
${AUTHOR_NAME}

加美元符, 花括号可加可不加

重新赋值

AUTHOR_NAME="Tony"
echo ${AUTHOR_NAME}
AUTHOR_NAME="Leung"
echo ${AUTHOR_NAME}

只读变量

AUTHOR_NAME="TonyLeung"
readonly AUTHOR_NAME

删除变量

unset AUTHOR_NAME

不能删除只读变量

字符串

单引号和双引号

单引号 双引号
字符原样输出, 变量无效 可以有变量
不能出现转义字符 可以有转义字符

获取字符串长度

echo ${#AUTHOR_NAME} #9

截取字符串

echo ${AUTHOR_NAME:4:5 #Leung

多行注释

:<<'
注释内容...
注释内容...
注释内容...
'

参数传递

参数处理 说明
$# 传递到脚本的参数个数
$* 以一个单字符串显示所有向脚本传递的参数。 如"$*"用「"」括起来的情况、以"$1 $2 … $n"的形式输出所有参数。
$$ 脚本运行的当前进程ID号
$! 后台运行的最后一个进程的ID号
$@ 与∗相同，但是使用时加引号，并在引号中返回每个参数。如"*相同，但是使用时加引号，并在引号中返回每个参数。如"∗相同，但是使用时加引号，并在引号中返回每个参数。如"@"用「"」括起来的情况、以"$1" "2"…"2" … "2"…"n" 的形式输出所有参数。
$- 显示 Shell 使用的当前选项
$? 显示最后命令的退出状态, 0 表示没有错误
