MYSQL安装

2019-09-27 2 min read # Mysql
  1. 进入官网
  2. DOWNLOADS - Community - MySQL Community Server
  3. 选择版本/位数
  4. 下载压缩包
  5. 解压
  6. 配置环境变量
path:%MYSQL%\bin
  1. 创建或修改%MYSQL%目录配置文件my.ini
    1. basedir=%MYSQL%
    2. datadir=%MYSQL%
    3. port=3306
  2. cmd(管理员 cd /)
cd %MYSQL%\bin
mysqld --initalize-insecure(初始密码为空)
mysqld -install
net start mysql
mysql -u root -p
  1. 卸载
net stop mysql
mysqld --remove mysql
  1. 更新root密码
set password for root@localhost = password('root');
  1. 忘记root密码
    1. 关闭正在运行的MySQL服务。
    2. 打开DOS窗口，转到mysql。
    3. 输入mysqld –skip-grant-tables 回车。–skip-grant-tables 的意思是启动MySQL服务的时候跳过权限表认证。
    4. 再开一个DOS窗口（因为刚才那个DOS窗口已经不能动了），转到mysql。
    5. 输入mysql回车，如果成功，将出现MySQL提示符 >。
    6. 连接权限数据库： use mysql; 。
    7. 改密码：update user set password=password(“root”) where user=“root”;（别忘了最后加分号） 。
    8. 刷新权限（必须步骤）：flush privileges;　。
    9. 退出 quit。
    10. 注销系统，再进入，重新登录
